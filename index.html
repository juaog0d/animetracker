⇄ Sincronizar via Git

Carregar JSON de uma URL (GitHub Raw)

Cole a URL raw do GitHub. Isso substituirá os dados atuais.

Fluxo de trabalho recomendado
1
Você (dono do repositório): edite os animes normalmente no site, depois clique em Exportar JSON. Um arquivo animes.json será baixado.
2
Substitua o animes.json antigo dentro da pasta do seu repositório local pelo novo que foi baixado.
3
No terminal, dentro da pasta do projeto: git add animes.json → git commit -m "update animes" → git push
4
Seu amigo: dentro da pasta do projeto dele: git pull. Ao abrir o index.html no navegador, os dados serão carregados automaticamente do animes.json atualizado.
5
Dica: Seu amigo também pode usar o campo de URL acima com o link raw do GitHub para carregar sem precisar clonar o repositório. O link fica no formato: raw.githubusercontent.com/usuario/repo/main/animes.json
